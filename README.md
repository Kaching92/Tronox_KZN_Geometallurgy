# 🏭 Tronox KZN Sands Integrated Operations Dashboard v5.0

**Powered by PraxiomAI.io**

## 📊 Overview

A comprehensive geometallurgical dashboard for real-time mining and metallurgical performance analytics at Tronox KZN Sands operations.

## ✨ Features

- **🏭 Mining Operations:** ROM, Waste, Ore, and HMC material flow tracking
- **🗺️ Geology Parameters:** THM, Slimes, TiO₂, ZrO₂ grade analysis with ore classification
- **⚙️ Plant Performance:** Recovery and throughput predictions based on ore characteristics
- **💰 Economic Summary:** Revenue, costs, NSR calculations with separate zircon value tracking
- **📈 Performance Analytics:** Smart recommendations engine for operational decisions
- **📥 Export:** Download all results as CSV for reporting
